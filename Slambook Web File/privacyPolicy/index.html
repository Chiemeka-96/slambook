APP MANAGER

Privacy Policy

App Manager do not collect personal information such as your name, email address, phone number or credit card. It can store file names, data and paths locally to use them later and for user's choice. It can store other configuration that could improve your experience when using the application. App Manager do not transfer this stored locally information outside of device.

Permissions

  • WRITE_EXTERNAL_STORAGE & READ_EXTERNAL_STORAGE: to read and write files on SD card.
  • INTERNET: to send reports about app's crashes and analytics using Firebase & show AdMob ads.
  • CLEAR_APP_CACHE : to clear installed app's cache.
